# Curso HCI — material de consulta

Este site reúne o conteúdo teórico e guias práticos do nosso curso de High-Content Imaging/Analysis (HCI/HCA).

## Como usar
- Navegue pelos módulos no menu à esquerda.
- Use a busca (ícone de lupa) para encontrar termos, pipelines e definições.
- Cada módulo tem leituras e “próximos passos” para as tarefas da semana.
