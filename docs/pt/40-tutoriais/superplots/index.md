# SuperPlots em Python 

Este tutorial apresenta uma estratégia de visualização que combina dados em nível celular com a variabilidade entre replicatas biológicas.

!!! note "Antes de começar"
    Para executar este tutorial, você precisa ter o [Pixi](https://pixi.sh/) instalado e utilizar um ambiente capaz de abrir notebooks, como o VSCode com a extensão Jupyter ou o JupyterLab.
    
    Também é aconselhável criar uma pasta e baixar os arquivos `combined.csv`, `pixi.toml` e `pixi.lock` na mesma pasta.

Os SuperPlots permitem mostrar, na mesma figura:

- a distribuição dos valores individuais;
- a média de cada replicata biológica;
- a reprodutibilidade entre experimentos independentes;
- a análise estatística baseada na unidade experimental correta.

O tutorial utiliza Python, Matplotlib, Seaborn, Pandas e SciPy.
