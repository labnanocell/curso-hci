# CellProfiler com RunCellpose

Este tutorial apresenta duas formas de instalar e executar o CellProfiler com o plugin RunCellpose: usando um ambiente Pixi reproduzível, como opção recomendada, ou Docker, como alternativa.

!!! note "Antes de começar"
    Para seguir a opção com Pixi, você precisa ter o [Pixi](https://pixi.sh/) e o [Git](https://git-scm.com/install/windows) instalados.

    No Windows, recomendamos utilizar o Git Bash para executar os comandos do tutorial.

    Caso você não tenha familiaridade com terminal ou prefira trabalhar com o CellProfiler Desktop instalado separadamente, o tutorial também apresenta uma alternativa usando Docker Desktop.

O tutorial mostra como:

- instalar e abrir o CellProfiler dentro de um ambiente Pixi;
- configurar corretamente a pasta `active_plugins`;
- carregar e executar o módulo RunCellpose;
- interpretar avisos comuns durante a inicialização;
- utilizar Docker como alternativa ao ambiente Pixi;
- testar se a instalação está funcionando corretamente.
