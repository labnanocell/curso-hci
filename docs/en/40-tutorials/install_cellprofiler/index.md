# CellProfiler with RunCellpose

This tutorial presents two ways to install and run CellProfiler with the RunCellpose plugin: using a reproducible Pixi environment, as the recommended option, or Docker, as an alternative.

!!! note "Before you begin"
    To follow the Pixi option, you need to have [Pixi](https://pixi.sh/) and [Git](https://git-scm.com/install/windows) installed.

    On Windows, we recommend using Git Bash to run the tutorial commands.

    If you are not familiar with the terminal or prefer to work with CellProfiler Desktop installed separately, the tutorial also presents an alternative using Docker Desktop.

The tutorial shows how to:

- install and launch CellProfiler inside a Pixi environment;
- correctly configure the `active_plugins` folder;
- load and run the RunCellpose module;
- interpret common warnings during startup;
- use Docker as an alternative to the Pixi environment;
- test whether the installation is working correctly.
