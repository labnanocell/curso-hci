# Tutorials

This section brings together practical materials for running analyses, organizing computational environments, visualizing data, and applying methods used in HCI/HCA.

The tutorials can be used independently or as a complement to the course’s conceptual modules.

## Available tutorials

### SuperPlots in Python

Learn how to simultaneously represent cell-level data and variability across biological replicates using Python, Matplotlib, and Seaborn.

Learn how to install and run CellProfiler with the RunCellpose plugin using a reproducible Pixi environment or, alternatively, Docker.
