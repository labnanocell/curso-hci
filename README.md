# Curso de HCI

This repository contains the source material for the **Curso de HCI** website.

The course was created primarily to train graduate students in the NanoCell Lab, but it is openly available for anyone who wants to read, use, or adapt the material.

![Course map](docs/assets/img/hci-aventura-mapa.webp)

## Access the course

The public course website is available at:

**https://labnanocell.github.io/curso-hci/**

If you arrived here looking for the course content rather than the repository itself, the link above is the best place to start.

## About this repository

This GitHub repository stores the course content in Markdown, along with the images and configuration files used to build the website.

Current structure:
- `docs/` — course pages and site content
- `docs/assets/` — images and static assets
- `mkdocs.yml` — MkDocs site configuration
- `requirements.txt` — Python dependencies

## Audience

The course is mainly designed for:
- graduate students working with high-content imaging/analysis;
- researchers interested in image-based assays, reproducibility, and data analysis;
- anyone who wants to explore the teaching material.

## Current status

The course is under active development and will continue to grow as new topics are added.

## Current topics

The course currently includes introductory material on:
- onboarding and scientific hygiene;
- scientific questions and operationalization;
- experimental design;
- batch effects;
- glossary and supporting material.

Additional modules will be added over time.

## Notes on reuse

Unless stated otherwise, the material is shared to support teaching and training.
